Documentation

Things used

For this project, I used Font Awesome icons for the footer icons (all four of them). I modified the hover to be more in line with my color scheme.
I also used Adobe Fonts to embed both of my fonts into each webpage and the CSS. This is done by linking to the styling page Adobe provided and then referencing the fonts in CSS as I would normally. I got to use Adobe Fonts through my Adobe Creative Cloud subscription.
I also looked at someone's guide for making a dropdown menu for the menu that shows up on my mobile design for this site. You can find his guide here.

Things that were hard

Pretty much everything regarding the dropdown menu was difficult, but I solved that with the guide I mentioned above. Besides that, the most difficult aspect of this project was working with flexbox. I'm not 100% used to it (more used to floats and clears), but it really sped up the process of making this site. For most of my issues, I referenced this guide. I found that it worked incredibly well as a "cheatsheet" for flexbox things I hadn't memorized.

Above & Beyond

I think my design went above my own expectations, especially considering I had a hard time figuring out how to include everything required and still maintain a style that I thought represented my aesthetics. I also think I modelled my resume page extremely similar to my actual resume, which was pretty difficult to do using flexbox considering my actual resume uses tables. Also, I liked my favicon.
