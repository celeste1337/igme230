NMD Poster Campaign
This was my final project for NMD II, a class that focuses on design fundamentals. This project was an exploration in color and shape for an imaginary poster campaign for safer sex. This project was important for me as someone who came from an abstinence-only state and an abstinence-only school to showcase how safe sex campaigns don't need to rely on fearmongering.
This project was made using Adobe Illustrator.
