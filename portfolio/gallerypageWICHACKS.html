WicHacks 2018
This was my group's final product for WicHacks 2018, an all-girls 24-hour hackathon. We made a grid-based game using p5, a Javascript library we had been using in class. The object of the game is to avoid the UFO while collecting stars. If the UFO touches you, you lose a health. At 0 health, the game ends! In our group of four, I was the second lead developer and implemented the grid for everything to "live" on, and the code for the UFO. You can check out the p5 documentation page here.
